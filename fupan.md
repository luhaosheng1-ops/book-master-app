| **环节**       | **技术重点**                 | **解决的坑点**                                     |
| -------------- | ---------------------------- | -------------------------------------------------- |
| **环境搭建**   | Docker + FastAPI             | 解决了 Python 环境配置难和跨平台运行问题。         |
| **文本提取**   | PyMuPDF + BeautifulSoup      | 攻克了 PDF、EPUB、TXT 多格式兼容的提取难题。       |
| **AI 调用**    | DeepSeek API + 结构化 Prompt | 解决了总结流于表面、干货不足的问题。               |
| **数据持久化** | Docker Volume + LocalStorage | 解决了容器重启文件丢失、浏览器刷新数据消失的问题。 |
| **项目协作**   | Git 代理 + GitHub 托管       | 攻克了国内访问 GitHub 频繁断开的连接难题。         |
